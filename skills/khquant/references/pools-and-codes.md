# 股票池与代码规范

## 股票代码格式

标准格式：`代码.市场后缀`

| 市场 | 后缀 | 代码前缀 | 示例 |
|------|------|---------|------|
| 上海 | `.SH` | 6, 5 开头 | `600000.SH`（浦发银行）|
| 深圳 | `.SZ` | 0, 3 开头 | `000001.SZ`（平安银行）|
| 北京 | `.BJ` | 4, 8 开头 | `430047.BJ` |

输入纯数字时 CLI 会自动补充后缀（如 `000001` → `000001.SZ`）。

## BaoStock 代码格式

BaoStock 使用 `市场.代码` 格式：
- `sh.600000`（上海）
- `sz.000001`（深圳）

CLI 会自动在内部转换，用户无需关心。

## 预设股票池

可用于 `--pool` 参数（`kh data download --pool hs300`）和 `kh tool pool` 命令。

| 池名 | 描述 | 典型用途 |
|------|------|---------|
| `a` | 沪深A股（全部）| 全市场回测 |
| `sha` | 上证A股 | 上证策略 |
| `sza` | 深证A股 | 深证策略 |
| `hs300` | 沪深300成分股 | 大盘蓝筹回测 |
| `sz50` | 上证50成分股 | 超大盘策略 |
| `zz500` | 中证500成分股 | 中盘策略 |
| `gem` | 创业板 | 成长股策略 |
| `star` | 科创板 | 科技股策略 |
| `etf` | 沪深ETF | ETF 策略 |
| `t0etf` | T0型ETF | 日内交易（T+0）|
| `t0stock` | T0股票型ETF | 日内交易（T+0）|
| `bond` | 沪深转债 | 可转债策略 |
| `index` | 常用指数 | 基准/对标 |

### 查看池内容

```bash
# 列出所有可用池
kh tool pool --all

# 查看某个池的具体股票
kh tool pool hs300
```

### 多池组合

```bash
# 同时下载沪深300和中证500
kh data download --source baostock --pool hs300,zz500 --period 1d
```

## 池文件位置

股票池 CSV 文件位于程序安装目录下的 `data/` 子目录。格式为 `代码.后缀,名称`，如：

```
000001.SZ,平安银行
600000.SH,浦发银行
```

使用 `kh data update-pool`（需要 xtquant）可更新成分股列表。

## 基准指数

回测默认基准为 `000300.SH`（沪深300指数）。每次执行 `kh data download` 时会自动检查并补充 000300.SH 的日线数据（2010年至今）。
